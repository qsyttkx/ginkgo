# ginkgo

一个简单的游戏框架(未完成)。

呃……按知乎里码农学生手写操作系统、编译器、游戏引擎的标准，我觉得还是有必要选一个尝试一下。再结合早期学长的蛊惑和自身兴趣爱好以及网上资料多寡的客观因素，我觉得写一个游戏框架应该相对<del>(容易)</del>有趣。

此前参加一个图标设计的投稿结果音讯全无蛮惨的，如今这个图标终于还是有了它的用武之地。

实际上这个就是把网上开源的库拼凑在了一起，没有多少自己的东西。甚至有些地方连进一步的封装都懒得封装了。
